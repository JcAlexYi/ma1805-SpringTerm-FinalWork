# Game title: "Disaster Game: Drop, Cover, Hold"
A surviving game in a disaster background,avoiding debris, collect coins to gain more points and collect first aid kit.

# Project Description
"Disaster Game: Drop, Cover, Hold" is a survival game set in the backdrop of natural disasters. Players control a character on the ground, dodging falling debris and collecting coins and first-aid kits. The goal is to collect items while dodging debris, with the difficulty increasing as the score rises.

The subtitle "Drop, Cover, Hold" is an international earthquake safety slogan. When players collect a first-aid kit, the game pauses and displays the message "Please check your first-aid kit regularly"—aiming to remind players of real-world emergency preparedness, using the game's dodging mechanics to convey realistic public safety information.

# How to Play
arrow keys to move
dodging falling debris, every game the player have 3 lives
collecting golden coins, which will add points
collecting first aid kits to pause and become immune to harm
survive as long as possible
press R to restart after Game Over
best score is saved automatically and shown on the game over screen
